Protocols

The following protocols are available globally.

Protocol Definition

  • StoryboardBased

    Make your ViewController subclasses conform to this protocol when:

    • they are Storyboard-based, and
    • this ViewController is the initialViewController of your Storyboard

    to be able to instantiate them from the Storyboard in a type-safe manner

    See more

    Declaration

    Swift

    public protocol StoryboardBased : AnyObject
  • StoryboardSceneBased

    Make your ViewController subclasses conform to this protocol when:

    • they are Storyboard-based, and
    • this ViewController is not the initialViewController of your Storyboard, but a different scene

    to be able to instantiate them from the Storyboard in a type-safe manner.

    You need to implement sceneStoryboard yourself to indicate the Storyboard this scene is from.

    See more

    Declaration

    Swift

    public protocol StoryboardSceneBased : AnyObject
  • NibLoadable

    Make your View subclasses conform to this protocol when:

    • they are NIB-based, and
    • this class is used as the XIB’s root view

    to be able to instantiate them from the NIB in a type-safe manner

    See more

    Declaration

    Swift

    public protocol NibLoadable : AnyObject
  • NibOwnerLoadable

    Make your View subclasses conform to this protocol when:

    • they are NIB-based, and
    • this class is used as the XIB’s File’s Owner

    to be able to instantiate them from the NIB in a type-safe manner

    See more

    Declaration

    Swift

    public protocol NibOwnerLoadable : AnyObject

Protocol definition

  • Reusable

    Make your TableViewCell and CollectionViewCell subclasses conform to this protocol when they are not NIB-based but only code-based to be able to dequeue them in a type-safe manner

    See more
