CollectionView

  • register(cellType:)

    Register a NIB-Based CollectionViewCell subclass (conforming to Reusable & NibLoadable)

    Seealso

    register(_:,forCellWithReuseIdentifier:)
  • dequeueReusableCell(for:cellType:)

    Returns a reusable CollectionViewCell object for the class inferred by the return-type

    Note

    The cellType parameter can generally be omitted and infered by the return type, except when your type is in a variable and cannot be determined at compile time.

    Seealso

    dequeueReusableCell(withReuseIdentifier:,for:)
